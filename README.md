
# COS202 Repository

## Assignments Completed

### 1. Mathematical Calculator (`calculator.py`)
- Interactive GUI calculator using Tkinter
- Supports: `+`, `-`, `*`, `/`, `^` (power), `%` (modulo), Clear, OFF

### 2. Personal Pocket CGPA Calculator (`cgpa_calculator.py`)
- Uses selection control statements (`if-elif-else`)
- Calculates CGPA and gives class classification (First Class, Second Class Upper, etc.)

## How to Run

```bash
python calculator.py
python cgpa_calculator.py
